Создай план разработки приложения и запиши его в отдельный файл. Вводные следующие:
1. Приложение должно работать офлайн на ПК
2. В качестве технологии будет использоваться Electron.
3. При нажатии на кнопку синхронизации - данные из приложения должны быть синхронизированными с удаленным сервером
4. В качестве UI фреймворка можно использовать ShadCN 
5. В качестве локальной БД где будут храниться данные можно использовать например Sqlite3 если это возможно

Структура приложения:
1. Главная. На главной находятся виджеты:
а). Тренировки сегодня
б). Должники(клиенты у которых не оплачены тренировки)
2. Клиенты
а). ФИО Клиента
б). Год рождения
в). Телефон
д). Дата последенй оплаты
е). Родители(для несовершеннолетних клиентов)
    - ФИО родителя
    - Телефон родителя
3. Группы
а). Название Групп
б). Дата старта Группы
в). Количество учеников
г). Расписание(Например понедельник с 20:00 - 21:30, среда с 20:00 - 21:30, пятница с 20:00 - 21:30)
д). Тренер(Назначается из сотрудников)

4. Занятия
а). Название Группы
б). Начало Занятия
в). Окончание Занятия
д). Иконка с отметкой посещаемости
Внутри занятия календарь соглано текущему расписанию группы с ячейками для каждого участника. Одна ячейка - одно
занятие. При клике на ячейку всплывает список: Был, Отсутствовал, Болеет и Удалить отметку если случайно проставил.

5. Сотрудники
а). ФИО
б). Год рождения
в). Телефон
В группы мы назначаем людей из списка сотрудников в качестве тренеров.
